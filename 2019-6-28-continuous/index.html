浅谈连续函数
发布于 2019-06-28   更新于 2019-09-02 数学 math 热度 0 喜欢 0

数学分析的学习过程中上来便是极限，连续的概念，而课本中的连续概念却恰恰是用极限定义的。如果细究数学史的话，连续概念是要先于极限的概念提出的。

以下内容旨在粗略介绍我眼中连续概念的前世今生。

连续初印象

  1. 从定义来说，连续就是函数在点 $x_0$ 处有 $\lim \limits_{x \rightarrow x_0}f(x) = f(x_0)$

  2. 闭区间上的连续函数具有很好的性质。最值定理、介值定理、康托定理、有界性等 $\cdots$

  3. 从邻域的角度理解连续，连续就是把开集映射到开集

连续和极限概念的提出

波尔查诺（Bernard Bolzano） 1817 年指出：若在区间内任意一点 $x$ 处，只要 $w$ 的绝对值充分小，就能使差 $f(x+w)-f(x)$ 的绝对值任意小，就说 $f(x)$ 在区间上是连续的。

Bourbaki, N., Elements d’histoire des mathematiques, Masson, Paris, 1984, ISBN 978-3-540-33938-0

其实连续最早的定义中已经看到了极限的雏形，后来柯西将这个和极限联系起来，其在1821 年的《教程》中说：当说及函数的连续性时，不需要说明无穷小量的性质，无穷小量是一个极限为 0 的变量。不过我们后来广泛使用的极限的定义是魏尔斯特拉斯提出的 $\epsilon-\delta$ 语言。

克莱因, 理京, 锦炎, 等. 古今数学思想[M]. 上海科学技术出版社, 2002.

可以说，极限和连续是一对孪生兄弟，是整个数学分析的基础。

简单整理连续概念的发展脉络如下：

  1. 1817 年波尔查诺提出连续性的概念，并给出了中值定理的证明。

  2. 1860 年魏尔斯特拉斯证明了最值定理。

  3. 1872 年海涅证明了实数系有界闭区间的连续函数一致连续。

    一般的，紧空间上的连续泛函为一致连续泛函。

连续的本质

连续有很多种刻画：

  1. 自变量变化比较小的时候，因变量变化也不大
  2. 极限值等于函数值本身
  3. 函数值与序列极限相等
  4. 实数系上连续与完备性等价（完备即柯西列)均收敛）
  5. 等等

不过如果要我说一种观点的话，我更倾向于认为连续是自变量与因变量之间的一种约束。光滑性则是更强的一种约束。

文章最开始举的例子中，连续函数仅仅首尾的函数值相同，便使得中间得到如此强的约束。

数学分析中我们讨论的最多的便是连续函数。而连续函数中我们讨论的最多的是什么？
